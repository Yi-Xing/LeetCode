## 序列求和

**链接**：http://lx.lanqiao.cn/problem.page?gpid=T2

### 题目

**资源限制**

&emsp;&emsp;时间限制：1.0s   内存限制：256.0MB

**问题描述**

&emsp;&emsp;求1+2+3+...+n的值。

**输入格式**

&emsp;&emsp;输入包括一个整数n。

**输出格式**

&emsp;&emsp;输出一行，包括一个整数，表示1+2+3+...+n的值。

#### 示例一：

**样例输入**

&emsp;&emsp;4

**样例输出**

&emsp;&emsp;10

#### 示例二：

**样例输入**

&emsp;&emsp;100

**样例输出**

&emsp;&emsp;5050

**数据规模与约定**

&emsp;&emsp;1 <= n <= 1,000,000,000。

### 解题思路

&emsp;&emsp;循环累加。
