## 77. 组合中等）

**链接**：https://leetcode-cn.com/problems/combinations/

### 题目

&emsp;&emsp;给定两个整数 n 和 k，返回 1 ... n 中所有可能的 k 个数的组合。

**示例:**

&emsp;&emsp;输入: n = 4, k = 2

&emsp;&emsp;输出:
````
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
````
